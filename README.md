# EZ :relieved:

Bootstrap projects to easily create applications with rich libraries support focused on DX.

## :computer: Usage

```bash
npx degit cristobalgvera/ez/{repository-folder} YOUR_APP_NAME
cd YOUR_APP_NAME

git init
yarn # npm i
```
